# Drum Corps International (DCI) Audio Quiz

A Python desktop application that plays short audio clips from Drum Corps International (DCI) shows and prompts the user to identify details such as the performing corps and the year of the show. The project is built with a graphical interface and modular quiz logic.

---

## Overview

This project is a quiz-style application designed to test users’ familiarity with DCI shows through audio recognition. The application selects a random clip from a predefined set of shows and asks the user to input identifying information. The logic, audio handling, and GUI are separated into distinct modules.

---

## Features

- Randomized selection of DCI show audio clips
- GUI built with Tkinter
- Input validation and answer checking
- Modular project structure (GUI, logic, audio handling separated)
- Audio playback management to prevent overlapping clips

---

## Technologies Used

- Python 3.13+
- Tkinter
- FFmpeg / ffplay for audio playback
- Object-oriented design for show metadata
